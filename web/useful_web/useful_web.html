Полезные веб-сайты

Учебные ресурсы по вёрстке веб-сайтов

Htmlbook.ru

Один из самых полезных сайтов по html и css. Этот ресурс создан в 2002 году. Для новичков будет полезен раздел «Шаг за шагом», в котором хорошо структурированы многое важные моменты.

html5book.ru

Сайт для тех, кто изучает веб-технологии и создает сайты.

Webref.ru

Один из самых полезных сайтов по html и css. На котором находится много полезной информации по верстке сайтов.

Dimox.name

Отличный блог Димокса, на котором можно найти массу материалов по продвинутому использованию css, верстке, а также по другим темам.

Ruseller.com

Css учебник и уроки по html и многим другим тематикам, полезным для вебмастера.

w3schools.com (HTML)

Подробный справочник по html на английском.

w3schools.com (CSS)

Подробный справочник по css на английском.

learn.javascript.ru (JS)

Перед вами учебник по JavaScript, начиная с основ, включающий в себя много тонкостей и фишек JavaScript/DOM на русском языке.

overapi.com (JS)

Шпаргалка по javascript на английском.

w3schools.com (JS)

Подробный справочник по javascript на английском.

Webdesign.org

Если вы хотите быть на волне технологии и быть в курсе всего самого нового в html и css, то этот сайт - для вас. Уроки обновляются с завидной регулярностью.

Habr.com (HTML)

Подборка интересных публикаций по html на хабре.

Habr.com (CSS)

Подборка интересных публикаций по css на хабре.

Habr.com (JS)

Подборка интересных публикаций по JavaScript на хабре.

Форумы и разделы форумов по вёрстке

Html.by

Полезный форум для вебмастера по html, css и многим другим тематикам.

Searchengines.guru

Раздел веб-строительство на серче - можно найти ответы не только на вопросы по html, но и на смежные темы.

Search Console

Google Search Console – это бесплатный сервис, с помощью которого вы можете получать информацию о том, как ваш сайт представлен в результатах поиска Google, оптимизировать контент и устранять возможные неполадки. Не обязательно регистрироваться в этом сервисе, чтобы ваш сайт попал в индекс Google. Однако это позволит узнать, как ваш ресурс сканируется нашими системами, и улучшить его взаимодействие с ними.

Совмещённый онлайн-валидатор W3C для проверки валидности html кода веб-страниц.

w3.org (CSS)

w3.org (RSS)

Caniuse.com

Онлайн-редакторы кода

Rapprogtrain.com

Редактор поддерживает HTML, CSS и JavaScript код.

Сodepen.io

Он предлагает поддержку HTML, CSS и JavaScript и огромного количества препроцессоров. Haml, Markdown, Slim и Jade поддерживаются, в качестве HTML-препроцессоров. Для CSS поддерживаются Less, SCSS, Sass и Stylus. Для JavaScript поддерживаются CoffeeScript, TypeScript, LiveScript и Babel.
Кроме этого, существует огромное сообщество разработчиков, использующих CodePen. Что позволяет легко найти демо-версии и примеры, созданные опытными разработчиками.

JSFiddle.net

Еще один популярный редактор CSS онлайн. Им успешно пользуются разработчики уже в течение довольно долгого времени, и он был первым до появления CodePen. JSFiddle — это простой в использовании онлайн-редактор кода с бесплатным совместным редактированием, текстовым и голосовым чатом. Вам даже не нужно регистрироваться, чтобы использовать функцию совместной работы.
JSFiddle также поддерживает SCSS и CoffeeScript. С помощью JSFiddle довольно просто совместно работать над кодом или размещать демо-версии.

Liveweave.com

Это online CSS редактор с функцией предварительного просмотра в режиме реального времени. Liveweave содержит встроенные контекстно-зависимые подсказки по коду HTML5, CSS3, JavaScript и JQuery. Он также позволяет загрузить ваш проект в виде архива, что очень удобно.
В Liveweave довольно просто подключать к проектам внешние библиотеки, такие как JQuery, AndgularJS, Bootstrap и т.д. Он также инструмент линейку, что помогает в разработке адаптивного веб-дизайна. Liveweave предлагает функцию «Team Up», которая предоставляет те же возможности, что и режим совместного редактирования в JSFiddle.

Полезные расширения для браузеров

Google Chrome

ColorZilla

Продвинутая пипетка, Color Picker, Gradient Generator и другие красочные вкусности ColorZilla, одно из самых популярных расширений для разработчиков Firefox с более чем 5 миллионами загрузок, наконец-то доступно для Chrome!
С ColorZilla вы можете получить цветное считывание из любой точки вашего браузера, быстро настроить этот цвет и вставить его в другую программу.

Code Cola

Инструмент для просмотра исходного кода страниц и редактирования CSS-кода. Можно менять тени, окружности бокса и так далее с помощью перетаскивания ползунка. После внесения изменений можно скопировать получившийся код и заменить его в коде сайта.

Chrome Sniffer

Расширение для браузера, которое определяет, какие JavaScript-библиотеки, фреймворк или CMS используются на ресурсе.

PerfectPixel

Верстка сайта пиксель-в-пиксель вызывает трудности как у новичков, так и у опытных специалистов. PerfectPixel накладывает изображение макета поверх верстки в браузере. В настройках расширения можно отрегулировать прозрачность и расположение макета, что позволяет учесть все необходимые размеры и отступы. Использование этого плагина упростит работу верстальщика и сократит количество правок при сдаче проекта. В моем наборе расширений PerfectPixel занимает первое место за счет своей функциональности и удобства использования.

Page Ruler

Простой инструмент, который помогает определить высоту, ширину и положение любого элемента на странице.

PageLiner

В графическом редакторе выровнять элементы или отобразить сетку сайта можно с помощью направляющих линий. Верстальщик может включить этот функционал в браузере используя PageLiner. Расширение отображает похожие направляющие, как в том же Adobe Photoshop. С PageLiner выравнивать элементы становится намного проще.

Project Naptha

Расширение для Google Chrome, которое позволяет выделять и копировать текст даже с картинок — будет полезным, по мнению Cretive Bloq, всем, кому хоть раз в своей работе приходилось иметь дело со встроенным текстом.

Ripple Emulator

Ripple Emulator — расширение-эмулятор для Google Chrome, которое позволяет тестировать веб-сайты на различных мобильных платформах с различными разрешениями экрана. Может быть использовано в сочетании с другими расширениями для тестирования и отладки ресурсов.

Search Stackoverflow

Расширение для быстрого поиска по популярному ресурсу для разработчиков Stack Overflow.

Tabsbook

Каждый работает с закладками по-своему. Есть те, кто сохраняет ссылки для быстрого доступа к сайтам, и те, кто используют закладки, чтобы хранить полезную информацию. С помощью Tabsbook можно превратить ваши закладки в базу знаний. Интерфейс расширения отображает древовидную структуру папок для удобной навигации по вашим закладкам. Все данные хранятся в облаке, что позволяет иметь доступ к ним с любого устройства.

Web Developer

Web Developer добавляет в браузер дополнительную панель с инструментами. Внутри нее находится 10 вкладок: Disable, Cookies, CSS, Forms, Images, Information, Miscellaneous, Outline, Resize и Tools. Функционал расширения позволяет:

  • отключать CSS, JS и изображения;
  • проверять валидность HTML, CSS;
  • манипулировать CSS стилями;
  • тестировать HTML формы;
  • отображать необходимую информацию мета тегов;
  • изменять размер окна браузера;
  • очищать кэш;

Web Developer остается популярным расширением среди разработчиков. Его можно сравнить с мультитулом в реальной жизни: всегда под рукой необходимый набор инструментов.

Web Developer checklist

Расширение для автоматической проверки, удовлетворяет ли сайт основным принципам SEO, достаточно ли он производителен и удобен для пользователя. Результаты проверки представляются в виде своеобразного чек-листа — можно посмотреть более подробную информацию и рекомендации по каждому из невыполненных пунктов, а также моментально исправить ошибки.

Wireframify

Дизайнеры проектируют сайты с помощью wireframe — это набор линий, блоков и подписей. Такой подход позволяет создавать архитектуру проекта с учетом отступов, размеров и расположения блоков на странице. Wireframify включает альтернативный вид сайта в виде wireframe, что позволит избежать ошибочного использования:

  • отрицательных margin;
  • выравнивания элементов с помощью padding;
  • добавления отступов с помощью CSS свойств left, right;

What Font

Расширение, которое позволяет мгновенно определить, какой шрифт использован на той или иной странице, не производя почти никаких дополнительных действий — наведя курсор мыши на надпись.

YSlow

YSlow — инструмент, который не только проверяет скорость загрузки той или иной веб-страницы, но и подсказывает разработчику, что её тормозит. Для этого расширение проверяет сайт на соответствие 23 из 34 правил производительности, сформулированных командой компании «Yahoo».

Mozilla Firefox

ColorZilla

Продвинутая пипетка, Color Picker, Gradient Generator и другие красочные вкусности ColorZilla, одно из самых популярных расширений для разработчиков Firefox с более чем 5 миллионами загрузок.
С ColorZilla вы можете получить цветное считывание из любой точки вашего браузера, быстро настроить этот цвет и вставить его в другую программу.
