О библиотеке

Онлайн библитека «Coda» (русс. - Кода), содержит в себе упорядоченные электронные копии книг по программированию, ссылки на полезные веб-ресурсы и полезное программное обеспечение для разработчика.
Электронные копии литературы, хранятся в форматах: .pdf, .djvu, .fb2 и доступны для скачивания с веб-хранилищ.

Для чтения вышеуказанных форматов книг, рекомендуется использовать следующие приложения просмотра документов (читалки):

  • Для Windows
    • «STDU Viewer» (.pdf,.djvu).
    • «AlReader» (.fb2).

  • Для GNU/Linux
    • «Evince» (.pdf,.djvu).
    • «FBReader» (.fb2).
